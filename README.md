# Convenience-Store-Infomation-App
편의점 할인 정보 제공 앱

네이티브 안드로이드 앱을 이용하여 편의점 할인 정보를 제공하는 어플리케이션  
파이썬을 이용하여 할인 정보를 크롤링 후 db에 저장  
jsp를 이용하여 안드로이드 앱이 db에 접근할 수 있는 웹 api 구현
